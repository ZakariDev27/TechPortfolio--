📁 File Organizer Pro

A modern Python desktop application that automatically organizes files with a clean UI and smart folder management.

---

Features
- Smart file organization by type
- Automatic folder creation
- Clean GUI with dropdown selection
- Progress bar tracking
- Real-time activity logs
- Fast and lightweight

---

Tech Stack
- Python 3.14
- Tkinter (GUI)
- OS Module (file handling)

---

How It Works
1. Select a file type from the dropdown
2. Click "Select Folder & Organize"
3. Files are sorted into folders like reports, logs, data
4. Progress updates in real time

---

Run the App
python file_organizer_pro.py
